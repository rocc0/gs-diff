## gs-diff1
Creates diff image from two images. Boundaries of both images should be identical.

###### Usage:
    go get -u github.com/rocc0/gs-diff
    go build .
    ./gs-diff -o=diff.png image1.png image2.png 
    or 
    go get -u github.com/rocc0/gs-diff/...
    gs-diff -o=diff.png image1.png image2.png
